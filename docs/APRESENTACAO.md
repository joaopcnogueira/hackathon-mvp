# Apresentação Hackathon 2026

## O Desafio AutoML

- Desenvolver uma plataforma de AutoML no-code capaz de:
- Realizar pré-processamento na base de dados 
- Treinar modelos de classificação e regressão
- Avaliar e exibir métricas adequadas
- Gerar artefatos para download
- Disponibilizar o modelo via API ou execução em batch
- Os times deverão utilizar os mesmos datasets


## Cronograma do Hackathon

- 02/02: Viagem de ida (manhã) + Abertura do Hackathon (tarde)
- 03/02: Desenvolvimento
- 04/02: Desenvolvimento
- 05/02: Desenvolvimento
- 06/02: Apresentação (manhã) + avaliação e premiação (tarde)
- 07/02: Viagem de retorno


## Entregáveis Esperados
- Plataforma No-code (point & click) functional e intuitiva
- Documentação mínima
- Demonstração do fluxo completo
- API disponível para teste durante avaliação
